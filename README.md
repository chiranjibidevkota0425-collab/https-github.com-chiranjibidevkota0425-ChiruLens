# ChiruLens - Professional Photo Editor

A modern, responsive web-based photo editor built with HTML, CSS, and JavaScript. Edit your photos with ease using intuitive controls.

## Features

✨ **Image Editing Tools**
- **Brightness** - Adjust image brightness from 0-200%
- **Contrast** - Enhance contrast from 0-200%
- **Saturation** - Modify color saturation from 0-200%
- **Blur** - Apply blur effect from 0-20px
- **Rotate** - Rotate images from 0-360 degrees

🎨 **User Experience**
- Real-time preview of all edits
- One-click reset to restore original settings
- Download edited images as PNG files
- Professional gradient UI with smooth animations
- Fully responsive design (desktop, tablet, mobile)

## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No backend required - runs entirely in the browser
